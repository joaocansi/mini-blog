### How to start

- Starting the postgres database:

  ```sh
    cd mini-blog
    docker-compose up
  ```
  
- Starting the server:

  ```sh
    cd mini-blog/server
    yarn start:dev
  ```
  
- Starting the client:

  ```sh
    cd mini-blog/client
    yarn dev
  ```
